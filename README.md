# Hangman — React + TypeScript

A modern, dark-themed Hangman game built with **React** and **TypeScript**.  
Smooth UI, clear visual feedback, a hint system, and a “Play Again” option make it fun and interactive.

---

## React

This project is built using **React 18** with **TypeScript** and **Vite**.  
It demonstrates a small interactive game with clean, modern UI/UX and smooth animations.

---

## Features

- **Random Word**: Each round selects a new random word.  
- **Letter Feedback**: Correct letters turn **green**, wrong letters turn **red**.  
- **Restart Game**: “Play Again” button appears at the end of the game.  
- **Hint System**: One hint per round that reveals a correct letter.  
- **Smooth UI**: Buttons have smooth animations and responsive design.  
- **Dark Theme**: Modern and visually appealing interface.  
- **Responsive Design**: Works well on both desktop and mobile devices.
